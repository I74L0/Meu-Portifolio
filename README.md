# 💻 Meu Portfólio Web

Este é o repositório do meu portfólio pessoal e profissional, desenvolvido para apresentar minhas principais habilidades, experiências, e projetos em destaque na área de desenvolvimento de software.

O design foi projetado com uma interface moderna, clean e responsiva, com suporte a efeitos visuais dinâmicos (efeito vidro/glassmorphism, gradientes modernos e micro-animações).

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias e bibliotecas:

*   **[React](https://react.dev/) (v19)** — Biblioteca principal para a construção da interface do usuário baseada em componentes reutilizáveis.
*   **[Vite](https://vitejs.dev/) (v8)** — Ferramenta de build extremamente rápida para o desenvolvimento front-end.
*   **[Tailwind CSS](https://tailwindcss.com/) (v4)** — Framework CSS utilitário para estilização rápida, responsiva e moderna.
*   **[Lucide React](https://lucide.dev/)** & **[React Icons](https://react-icons.github.io/react-icons/)** — Conjuntos de ícones modernos e minimalistas para a interface.

---

## ✨ Funcionalidades Principais

*   **Seção de Apresentação (Hero):** Banner inicial dinâmico contendo links diretos para redes sociais (GitHub, LinkedIn, E-mail) e foco de atuação.
*   **Sobre Mim (About):** Breve biografia detalhando minha trajetória acadêmica e profissional.
*   **Experiência (Experience):** Linha do tempo apresentando histórico profissional recente.
*   **Projetos em Destaque (Projects):** Cards dinâmicos e com efeito *hover zoom* integrando imagens reais dos projetos:
    *   *PetCare+* — Aplicativo móvel para gestão pet (Figma & Flutter).
    *   *Gerenciador Jotanunes* — Painel Web fullstack (React & Django).
    *   *Meu Sócio* — Planejamento e design de app financeiro MEI (Figma).
*   **Habilidades (Skills):** Cards organizados com o nível de experiência/domínio de tecnologias (Python, Django, React, C#, Flutter, etc.).
*   **Formulário de Contato:** Área de contatos diretos (WhatsApp e E-mail dinâmicos) e formulário interativo de envio de mensagens.

## 🚀 Como Executar o Projeto Localmente

Siga os passos abaixo para rodar o projeto em sua máquina de desenvolvimento:

### Pré-requisitos
Você precisará ter o [Node.js](https://nodejs.org/) instalado em seu computador (recomenda-se versão LTS).

### 1. Clonar o repositório
```bash
git clone https://github.com/I74L0/Meu-Portifolio.git
```

### 2. Acessar a pasta do projeto
```bash
cd meu-portifolio-web
```

### 3. Instalar as dependências
```bash
npm install
```

### 4. Iniciar o servidor de desenvolvimento
```bash
npm run dev
```
O projeto estará disponível por padrão no endereço: `http://localhost:5173/`

---

## 📦 Build para Produção
Para gerar a pasta `dist` com o código otimizado para o deploy:
```bash
npm run build
```
